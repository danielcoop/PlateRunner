# PlateRunner for Umbraco

**PlateRunner** is an open-source Umbraco package that helps small restaurants, food trucks, and cafes create branded microsites with editable menus.

It is also the foundation for a hosted service (PlateRunner Cloud) for customers who want an out-of-the-box hosted experience.

## What PlateRunner Does

- Branded public microsites
- Editable menu sections and menu items
- Specials / announcements
- Opening hours
- Contact details
- Social links
- QR-friendly mobile pages
- Built-in visual themes

## Initial Themes

- Minimal Elegant
- Retro Diner
- Fiesta Street

## Intended Users

- Umbraco developers
- Agencies building hospitality sites
- Freelancers
- Self-hosting businesses
- Future hosted customers

## MVP Scope

Version 1 includes:

- Package installation into an Umbraco site
- Microsite structure
- Menu sections and items
- Featured announcements / specials
- Opening hours, contact info, social links
- Three built-in themes
- Sample/demo setup
- Documentation

## Non-Goals

- Online ordering
- Payments
- POS integrations
- Delivery integrations
- Loyalty programs
- Advanced analytics
- Inventory management
- Multi-tenant SaaS administration

## Architecture

- Package-first
- Razor-based rendering
- Theme resolver abstraction
- View model mapping layer
- Simple service boundaries
- Standard Umbraco conventions

## Sample Site

A sample Umbraco site is included to demonstrate:
